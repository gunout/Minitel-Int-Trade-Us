# Minitel-Int-Trade-Us
MINITEL INTERFACE - US STOCK TRADER  with SPACE X  on localhost:5001 . Propulsed by YFINANCE .

# FOR LINUX 
1 . INSTALL PYTHON VENV     
    
    python -m venv mini 

    source mini/bin/activate

2 . INSTALL REQUIREMENTS :

    pip install -r requirements.txt
# RUN INTERFACE

    python serv.py

# FOR WINDOWS 

1 . INSTALL PYHTON VENV 

    python -m venv mini 

    mini\Scripts\activate.bat

2 . INSTALL REQUIREMENTS 

    pip install -r requirements.txt 

# RUN INTERFACE 

    python serv.py
